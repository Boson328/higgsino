## 概要

このライブラリはタイピングソフト（特にローマ字タイピングソフト）の制作者を手助けするために作られました。

導入することで以下のようなことが簡単に行えるようになります。

- ひらがなからローマ字への変換
- タイピング動作を再現
- すべての打ち方対応 ( 例: ゆっくりじっきょう → yuxtsucurijikkilyou )
